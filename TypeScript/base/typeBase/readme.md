## TS

为什么要学？

特点： 静态的（编译时报错），类型的。有利于多人协作，代码可读性。

严谨但不失灵活

#### 缺点：
- 与实际框架结合会有很多坑
- 配置学习成本高
- TypeScript类型系统比较复杂

### 基础类型：base.ts

- any 多人协作项目的大忌，不应该首先考虑此类型
- unknown 与any不同之处： unknown类型被确定是某个类型之前，不能被进行任何操作，如实例化，getter，函数执行等

#### 数组：
- 泛型定义： 
```js
const list: Array<number> = [1, 2, 3]
```
- []:
```js
const list: number[] = [1, 2, 3]
```
